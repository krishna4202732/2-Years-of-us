# HTML CSS JavaScript Anniversary Website

This is the standalone version of the anniversary website. It uses only:

- `index.html`
- `styles.css`
- `script.js`

Open `index.html` directly in a browser, or serve this folder with any static server.

All images are intentionally left as empty placeholders with `src=""`. Replace the empty values in `index.html` for gallery images and in `script.js` for timeline images.
